# Change log — Vertical Pinch-to-Zoom

Implements plan `plans/20260716_073540_vertical-pinch.md`.

## The problem

Pinch-to-zoom was responding to scale changes along both horizontal and vertical axes. Users required that pinch-to-zoom respond only to vertical gestures (pinching up and down) while ignoring horizontal gestures.

## What changed

`lib/core/editor/pinch_to_zoom_area.dart`:
- Updated `onScaleUpdate` in `PinchToZoomArea` to compute the new font scale using `details.verticalScale` instead of `details.scale`.

`test/shell/tabs/pinch_to_zoom_test.dart`:
- Added a new unit test `PinchToZoomArea ignores horizontal pinch gestures but accepts vertical ones` that simulates both horizontal-only and vertical-only pinch gestures and asserts that the font scale only updates in response to vertical gestures.

## Verification

- Ran unit tests: `flutter test test/shell/tabs/pinch_to_zoom_test.dart` (Passed)
- Ran full unit test suite: `flutter test` (All 562 tests passed successfully)
