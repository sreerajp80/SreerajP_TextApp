# Remove the unused homeSyncTooltip localization string

**Status:** completed

## The issue

After removing the sync icon from the Recent files toolbar, the localization
string `homeSyncTooltip` ("Sync with another device") is no longer used. The
user does not want stale code left behind, so this string should be removed.

## The plan for the fix

1. In [lib/l10n/app_en.arb](../lib/l10n/app_en.arb): delete the two lines for
   `homeSyncTooltip` (the value line and its `@homeSyncTooltip` description line).
2. Regenerate the localizations with `flutter gen-l10n`. This updates the two
   generated files:
   - `lib/l10n/app_localizations.dart` (removes the abstract getter)
   - `lib/l10n/app_localizations_en.dart` (removes the English getter)
3. Confirm no other code references `homeSyncTooltip`.

## Files to be changed

- `lib/l10n/app_en.arb` — remove the string and its description (hand-edited).
- `lib/l10n/app_localizations.dart` — regenerated by `flutter gen-l10n`.
- `lib/l10n/app_localizations_en.dart` — regenerated by `flutter gen-l10n`.

## Testing

- `flutter analyze` — no issues.
- `flutter test test/l10n/localization_test.dart` — passes.

## Notes

- Only the generator touches the two `app_localizations*.dart` files, keeping the
  arb and generated code in sync.
