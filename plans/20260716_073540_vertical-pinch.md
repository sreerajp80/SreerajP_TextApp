# Vertical Pinch-to-Zoom

**Status:** completed

## The issue

Currently, the pinch-to-zoom gesture responds to both horizontal and vertical scale changes. The requirement is that pinch zoom-in and zoom-out should work only vertically, completely ignoring horizontal gestures.

## Files to change

- [lib/core/editor/pinch_to_zoom_area.dart](file:///l:/Android/SreerajP_TextApp/lib/core/editor/pinch_to_zoom_area.dart)
- [test/shell/tabs/pinch_to_zoom_test.dart](file:///l:/Android/SreerajP_TextApp/test/shell/tabs/pinch_to_zoom_test.dart)

## The fix

1. Update `PinchToZoomArea` in `lib/core/editor/pinch_to_zoom_area.dart` to use `details.verticalScale` instead of `details.scale` when updating the font scale.
2. Update the unit tests in `test/shell/tabs/pinch_to_zoom_test.dart` to:
   - Confirm that vertical movement updates the font scale as expected.
   - Add a test verifying that horizontal-only movements do not alter the font scale.

## Testing

- Run unit tests: `flutter test test/shell/tabs/pinch_to_zoom_test.dart`
- Run all project tests: `flutter test`
